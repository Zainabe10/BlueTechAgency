# Projet Tailwind CSS – BlueTech Agency

**Contexte :**  
Vous êtes stagiaire chez **BlueTech Agency** et votre mission est de **refaire leur site web vitrine** en utilisant le framework **Tailwind CSS**. L’objectif est de reproduire le design fourni à l’aide des **captures d’écran du site attendu**, qui sont disponibles dans le PDF joint.
